Return Largest Numbers in Arrays


Basic Algorithm Scripting in javascript

Challenge: Return an array consisting of the largest number from each provided sub-array.

Remember, you can iterate through an array with a simple for loop, and access each member with array syntax arr[i] .

Share this Challenge 


        //find largest numbers of four sub-arrays
        function largestOfFour(arr) {
        var largestNum;
        var largestNumArray = new Array();
        //loop throu,find largest, push it to the new Array
        for(var i = 0; i< arr.length ; i++){
        largestNum = arr[i].sort(function(a,b){return b-a;}[0];
        largestNumArray.push(largestNum);
        }
        return largestNumArray;
        }
        //run the function
        largestOfFour([[4, 5, 1, 3], [13, 27, 18, 26], [32, 35, 37, 39], [1000, 1001, 857, 1]]);
